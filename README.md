# Customer Shopping Behaviour Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a customer shopping behaviour dataset using Python.

## Objectives
- Clean and preprocess the dataset
- Handle missing values
- Perform descriptive statistics
- Visualize trends and patterns
- Generate business insights

## Technologies Used
- Python 3.12
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure
```
.
├── Customer_shopping_behaviour.ipynb
├── customer_shopping_behavior.csv
├── requirements.txt
├── .gitignore
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

Open the notebook:

```bash
jupyter notebook
```

and run `Customer_shopping_behaviour.ipynb`.

## Dataset
The dataset contains customer purchase information used for exploratory data analysis and visualization.
